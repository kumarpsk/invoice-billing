# Supermarket POS Backend

## Prerequisites
- Java 17
- Maven 3.9+
- PostgreSQL 16

## Run database
```bash
docker compose up -d
```

## Run application
```bash
mvn spring-boot:run
```

## Swagger
Open: `http://localhost:8080/api/swagger-ui`

## Default admin user
- Username: `admin`
- Password: `admin123`
